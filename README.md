### Реализованные игры:
1. Игра на скорость реакции  
   - Случайное появление кнопки  
   - Таймер 30 секунд  
   - Подсчёт нажатий и среднего времени реакции  

2. Крестики-нолики  
   - Поле 3×3  
   - Поочерёдная смена игроков  
   - Проверка победы и ничьей  
   - Перезапуск игры  

3. Лабиринт (опционально)  
   - Управление стрелками  
   - Таймер прохождения  
   - Проверка достижения выхода
